# Python Security Tools 🐍🔐

Coleção de scripts em Python voltados para segurança da informação e automação de tarefas de pentest.

## 📌 Objetivo

Este repositório contém ferramentas simples e scripts utilitários desenvolvidos em Python para auxiliar em atividades de segurança ofensiva e defensiva.


## 🧰 Ferramentas incluídas

### 🔎 Port Scanner
- Scanner básico de portas TCP
- Identificação de serviços abertos

### 🔐 Brute Force Tools
- Testes de autenticação (educacional)
- Wordlists simples

### 🧾 Hash Utils
- Geração e verificação de hashes (MD5, SHA256, etc.)

### 🌐 Network Sniffer
- Captura básica de pacotes com Scapy

### 🕸️ Web Checker
- Verificação de headers HTTP
- Testes simples de endpoints

## ⚙️ Requisitos

- Python 3.8+
- Bibliotecas:
  - `socket`
  - `hashlib`
  - `scapy` (opcional)
  - `requests`

Instalação:
```bash
pip install -r requirements.txt
